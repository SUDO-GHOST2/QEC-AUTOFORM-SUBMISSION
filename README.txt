# QEC Autoform Submission
This project automates the submission of QEC feedback forms  using cURL in C++.

## Features
- Automatically logs in to the portal.
- Extracts dropdown options for teachers and subjects.
- Submits feedback for all options.

## Prerequisites
- Install `libcurl`.
- C++ compiler (e.g., GCC).

## Usage
- Compile the program using a C++ compiler.
- Run the executable and follow the prompts to enter your credentials.
